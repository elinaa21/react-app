 # Messenger
![alt-текст](./src/img/chat.jpg)


## Technology stack
- React
- Redux
- SCSS
- Webpack
- Node.js
- MongoDB
- Socket.IO

## Oportunities
- Sending and receiving messages
- Authorization
- Registration

## Future improvements
- Adding to favorites (button star)
- Сontact search

## Launch
1. install mongoDB
2. create database 'test', create collections 'sessions' and 'dialogs'
3. npm i
4. npm run server
5. npm start
6. check localhost:8080
